# Employee Management Dashboard

## Project Overview

This is a React.js-based Employee Management Dashboard application that provides a simple way to manage employee records. It includes a mock login authentication flow, a dashboard showing employee summary data, and a detailed employee list with options to add, edit, delete, and print employee information.

## Features

- **Authentication**
  - Mock login page
  - Protected dashboard access (cannot access without login)

- **Dashboard**
  - Displays total number of employees
  - Shows active vs inactive employee counts

- **Employee List**
  - Columns: Employee ID, Profile Image, Full Name, Gender, DOB, State, Active/Inactive toggle
  - Actions for each employee: Edit, Delete (with confirmation), Print

- **Employee Form**
  - Add or edit employee details
  - Fields: Full Name, Gender, DOB, Profile Image upload (with preview), State (dropdown), Active/Inactive status
  - Form validation included

- **Search & Filter**
  - Search employees by name
  - Filter by gender and active/inactive status
  - Filters work in combination

- **Print**
  - Print employee list functionality

- **UI/UX**
  - Clean and modern design
  - Responsive layout
  - Loading and empty states handled gracefully

## Technology Stack

- React.js (JavaScript)
- React Router for navigation
- CSS for styling (or CSS Modules / Styled Components)
- Mock API using localStorage
